# SampleCo — deliberately vibe-coded demo app

A fictional "invoice + user" API built to look like a rushed AI-generated MVP. It contains
**intentional, documented defects** (see PLANTED_DEFECTS.md) spanning the CodeReckon taxonomy:
AI-slop, security, dependency, and maintainability issues, plus a multi-step attack chain.

Two purposes:
1. **Flagship sample report** — the public CodeReckon Audit Report is generated from this repo.
2. **Pipeline validation** — every planted defect must be caught by the audit pipeline; a miss
   is a pipeline bug (red-team-your-own-tool discipline).

NOT a real product. Do not deploy. Secrets/keys here are fake.
